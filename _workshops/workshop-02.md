# Purpose
To build comfort using the command line and the Boston University's Shared Computing
Cluster. Both of these will be integral in all of the following workshops.

# Skills
- Navigating a filesystem and manipulating files
- Writing and running basic shell scripts
- Submitting and monitoring jobs on the SCC

# Knowledge
Core syntax, file permissions, standard Unix tools, SGE Job Scheduler


# Tasks



# Criteria for Success
You can successfully navigate across shared file directories on the SCC. You 
understand how to check file permissions and how to modify them. You can run
and write a basic shell script by hand. You can use `qsub` and `qstat` to check
on the status of jobs you've submitted to the cluster.
